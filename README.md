# 📱 QR Code Generator (Flask)

## 📌 Project Overview

This project is a web-based **QR Code Generator** built using **Python Flask**.
It allows users to generate QR codes for text, URLs, or any input data instantly.

---

## 🚀 Features

* Generate QR codes dynamically
* Supports text, URLs, and custom input
* Lightweight Flask backend
* Simple and clean UI

---

## 🛠️ Tech Stack

* Python
* Flask
* HTML
* CSS

---

## ⚙️ How It Works

1. User enters text or URL
2. Flask backend processes input
3. QR code is generated using Python library
4. QR image is displayed on the webpage
